## First-time Setup

### TextMeshPro

  * Window > Package Manager > Plus Dropdown > Add package by name...
    * Name: `com.unity.textmeshpro`
    * Version: `3.2.0-pre.6`
  * Window > TextMeshPro > Import TMP Essential Resources

### DOTween

  * DOTween Utility Panel
    * Setup DOTween
    * Create ASMDef
